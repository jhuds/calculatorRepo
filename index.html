Calculator Functionality

This calculator in current version offers limited functionality. It prompts for two Values and an arithmetic operation to be performed between them. The functional steps are as follows:

  1. Prompt for Value 1

  2. Prompt for arithmetic opeation to perform

      +      -      *       /        %       1/x    square root

  3. Prompt for optional Value 2

Depending on the arithmetic operator chosen such as: \[\sqrt{}\] \[\frac{1}{x}\] the application ignores the second value and applies the calculation just using first value.

Calculator Shiny App Components

The Calculator Shiny app build requires two components: ui.R and server.R scripts with R code.

  • ui.R: prompts to accept user inputs and display results rendered by server component
  • server.R: process the input values, apply the arithmetic operation and return the result

e.g. Upon prompts if user entered values have following content value1 <- 91, value2 <- 10 and operator <- '+' and '1/x'. The results are as follows:

value1 <- 91;  value2 <- 10; operator <- '+';
eval(parse(text=paste(as.character(value1), operator, as.character(value2))))

## [1] 101

eval(parse(text=paste('1 / ', as.character(value1))))

## [1] 0.01098901
